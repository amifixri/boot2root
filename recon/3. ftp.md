# FTP Enumeration

Tujuan:

Mencari file yang dapat diakses.

---

## Login Anonymous

```bash
ftp TARGET
```

Username:

```text
anonymous
```


## List File

```bash
ls
```

---

## Download File

```bash
get file.txt
```

---

## Download Semua File

```bash
mget *
```
